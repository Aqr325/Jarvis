## Jarvis Agent 性能优化与推理升级 (第二轮)
- 新增三个核心中间件模块：
  - `core/rate_limiter.py` — 令牌桶算法，per-client/per-endpoint 限速
  - `core/timeout.py` — 响应超时控制 + 熔断器模式 (Circuit Breaker)
  - `core/websocket_manager.py` — WebSocket 连接管理、广播、流式响应、心跳检测
- 升级 `core/engine.py`：
  - 新增 `DecisionStateMachine` — 5 状态管道 (IDLE -> PERCEIVING -> REASONING -> EXECUTING -> REMEMBERING)
  - 新增 `ContextWindowManager` — 滑动窗口上下文管理 + 关键词相关性检索
  - 新增 `ReasoningStep` / 多步推理链 (intent -> context -> KG -> synthesis)
  - 新增 `DecisionStrategy` 枚举: NLP_ROUTE / STATE_MACHINE / MULTI_STEP / RULE_BASED / FALLBACK
- 升级 `server.py`：
  - 集成 rate_limiter, timeout_manager, websocket_manager
  - 新增 `/ws` WebSocket 端点（流式打字效果、工具执行进度推送、ping/pong）
  - 新增 `/api/admin/circuit-breaker` 监控端点
  - 每个 API 端点通过 decorator 注入速率限制和超时控制
- 更新 `requirements.txt` 增加 `websockets>=12.0` 和 `httpx>=0.25.0`
